РУКОВОДСТВО ПО ЭКСПЛУАТАЦИИ
	База данных поддерживает возможности:
1.	записи новых строк таблиц;
2.	создание различных профилей со своими разрешениями;
3.	удаление строк таблиц; 
4.	осуществление выборки данных по таблицам.
Для записи новых строк воспользуйтесь командой: 
- INSERT INTO Название_таблицы (Поля_таблицы)   VALUES (Значения);
где Поля_таблицы — столбцы, которые вы хотите заполнить(ввод через запятую), Значения — данные, которые вы хотите занести в выбранные столбцы(ввод в том же порядке, что и Поля_таблицы, через запятую).
Для удаления строк таблицы воспользуйтесь командой: 
- DELETE FROM Название_таблицы WHERE Код_строки;
Для создания различных профилей воспользуйтесь командой:
- CREATE USER 'Имя_пользователя'@'localhost' IDENTIFIED BY 'Пароль';
Затем введите следующую команду для выдачи прав новому пользователю:
- GRANT Тип_прав ON Имя_базы_данных.Имя_таблицы TO ‘Имя_пользователя’@'localhost’;
где Тип_прав вводится через запятую из списка ниже: 
1)	CREATE — Позволяет пользователям создавать базы данных/таблиц;
2)	SELECT — Разрешает делать выборку данных:
3)	INSERT — Право добавлять новые записи в таблицы;
4)	UPDATE — Позволяет изменять существующие записи в таблицах;
5)	DELETE — Даёт право удалять записи из таблиц;
6)	DROP — Возможность удалять записи в базе данных/таблицах;
7)	ALL PRIVELEGES — все права, перечисленные выше.
